# Portfolio React (Vite)

Projeto gerado automaticamente — versão modularizada do `portfolio.html`.

## Como usar

1. Instale dependências:
   ```bash
   npm install
   ```
2. Rode em desenvolvimento:
   ```bash
   npm run dev
   ```

> Nota: Tailwind está sendo carregado via CDN no `index.html` para acelerar o setup.
